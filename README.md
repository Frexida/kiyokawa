# 清川商店 公式サイト

福島県喜多方市で最も古い酒蔵「清川商店」の公式Webサイト。

## 概要

- **蔵元名:** 会津清川醸造元 清川商店
- **創業:** 寛永8年（1631年）
- **所在地:** 福島県喜多方市二丁目4659
- **特徴:** 喜多方最古の酒蔵、女性杜氏、古酒（12年・15年・18年熟成）

## サイト構成

- `index.html` - トップページ
- `products.html` - 商品紹介（古酒シリーズ、定番商品）
- `about.html` - 蔵について（歴史、女性杜氏、蔵見学案内）

## 技術スタック

- HTML5 + Tailwind CSS
- Cloudflare Pages
- Google Fonts (Noto Serif JP)

## 開発

```bash
# ローカルプレビュー
npx wrangler pages dev site

# デプロイ
npx wrangler pages deploy site
```
